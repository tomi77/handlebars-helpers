# handlebars-helpers

A set of Handlebars helpers

## string helpers

* nl2br - replace new line characters with <br/>

## date helpers

* moment - wrapper to moment.js

## logical helpers

* coalesce - return first not empty value
* eq
* neq
* lt
* lte
* gt
* gte
